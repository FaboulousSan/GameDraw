# GameDraw
GameDraw resout la paralysie du choix face a une ludotheque trop fournie : un clic, un jeu tire au sort dans ta bibliotheque (Switch, PC, ou toute autre plateforme que tu ajoutes), avec un roulement equitable (chaque jeu passe une fois avant qu'un autre repasse - meme lui, oui, meme celui que tu as achete en solde et jamais lancé !
